# 2025-01-otus-java-basic-console-chat

1. ДЗ 15: Реализуйте возможность отправки личных 
сообщений: если клиент пишет «/w tom Hello», 
то сообщение Hello должно быть отправлено только 
клиенту с ником tom

2. ДЗ 16: Добавьте пользователям роли: USER, ADMIN
   Для пользователей с ролью ADMIN реализуйте 
   возможность отключения пользователей от чата с 
   помощью команды «/kick username»

3. ДЗ 18: Добавить в сетевой чат аутентификацию через 
   базу данных SQLite (При желании можете взять другую СУБД)

Дополнительно: 

Усложните серверную часть чата так, чтобы она автоматически проверяла текст сообщений,
поступающих от пользователей, на предмет наличия запрещенных слов (список которых
подгружается сервером при запуске из файла). При обнаружении запрещенного слова в сообщении
сервер должен проигнорировать сообщение (не рассылать его другим пользователем) и разорвать
соединение с данным пользователем.

Усложните серверную часть простейшего чата так, чтобы она следила за количеством
подключенных пользователей и не позволяла подключаться большему числу пользователей,
нежели 5.
